# Параллельные миры

Интерактивная сравнительная шкала цивилизаций, государств, культур, религий и философских традиций от 3500 года до н. э. до 1600 года н. э.

**Сайт:** [Русская версия](https://agent-axiom.github.io/parallel-worlds/?lang=ru) · [English version](https://agent-axiom.github.io/parallel-worlds/?lang=en) · [简体中文](https://agent-axiom.github.io/parallel-worlds/?lang=zh)

## Возможности

- 49 исторических линий, 196 периодов и 147 хронологических ориентиров;
- Шумер, Египет, Византия, Индия, Китай, Корея, Япония, Юго-Восточная Азия, Океания, майя, ацтеки, инки и другие общества;
- 18 религиозных и философских традиций;
- полнотекстовый поиск, фильтры по региону и типу;
- готовые диапазоны, масштабирование и срез выбранного года;
- подробные карточки и обзорные источники;
- светлая и тёмная темы, URL-состояние и экспорт CSV;
- адаптивный интерфейс без runtime-зависимостей.
- полная русская, английская и китайская локализация интерфейса и исторических данных.

Рекомендации по следующим локалям описаны в [языковой стратегии](docs/LANGUAGE_ROADMAP.md).

## Локальный запуск

```bash
npm test
npm run serve
```

Откройте `http://localhost:8080/`. Для просмотра можно также открыть `index.html` напрямую.

## Проверка

```bash
bash scripts/validate.sh
```

Проверяются синтаксис JavaScript, схема и полнота данных, чистые функции шкалы, наличие deployment-файлов и относительные пути ресурсов.

## Публикация

Push в `main` запускает [GitHub Actions workflow](.github/workflows/deploy-pages.yml), который проверяет сайт и публикует статический артефакт в GitHub Pages.

## Методика

Границы периодов используются как крупные сравнительные ориентиры. Они не означают мгновенного возникновения или исчезновения культур. В интерфейсе приведены ссылки на обзорные хронологии The Metropolitan Museum of Art, British Museum, Smarthistory, Asia Society и UNESCO.

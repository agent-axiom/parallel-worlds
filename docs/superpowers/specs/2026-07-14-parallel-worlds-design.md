# «Параллельные миры»: дизайн сайта

## Цель

Создать автономный русскоязычный сайт, который позволяет сравнивать цивилизации, государства, культуры, религии и философские традиции на общей шкале от 3500 года до н. э. до 1600 года н. э. Сайт должен публиковаться в `agent-axiom/parallel-worlds` через GitHub Pages.

## Интерфейс

Главный экран сочетает вводную область, компактную панель управления и горизонтальную сравнительную шкалу. У каждой линии есть регион, тип, краткое описание, периоды и ключевые события. Цвет кодирует регион, вертикальное положение — отдельную историческую линию, а горизонтальное — время.

Пользователь может:

- искать по линиям, периодам, событиям и описаниям;
- фильтровать данные по региону и типу;
- выбирать готовые временные диапазоны;
- менять масштаб и тему;
- перемещать указатель года и видеть современников;
- открывать подробную карточку линии;
- экспортировать текущую выборку в CSV;
- делиться URL, который сохраняет фильтры и диапазон.

На мобильном устройстве панель управления перестраивается в один столбец, а шкала прокручивается внутри собственного контейнера, не создавая горизонтальную прокрутку всей страницы.

## Архитектура

Проект не требует сборки и runtime-зависимостей. `index.html` задаёт семантическую структуру, `styles.css` — адаптивную визуальную систему, `data.js` — исторический набор данных, `timeline.js` — чистые функции состояния и геометрии, `app.js` — DOM-контроллер.

Данные экспортируются в браузер через `window.PARALLEL_WORLDS_DATA`, а чистые функции — через UMD-совместимый объект. Это позволяет тем же файлам работать напрямую на GitHub Pages и проверяться локальным Node.js 12 без транспиляции.

## Данные и допущения

Диапазоны отражают удобную сравнительную периодизацию, а не утверждение о мгновенном возникновении или исчезновении культуры. Приблизительные и традиционные даты отмечаются в данных. Сайт содержит ссылки на обзорные хронологии музеев и энциклопедий; интерфейс прямо сообщает, что границы условны.

## Надёжность и публикация

Локальная проверка валидирует схему данных, границы периодов, уникальность идентификаторов, фильтрацию, преобразование координат и обязательные файлы Pages. GitHub Actions запускает те же проверки, формирует статический артефакт и публикует его в environment `github-pages`.

## Критерии приёмки

1. Сайт открывается без сборки и внешних JavaScript-библиотек.
2. Присутствуют не менее 49 исторических линий, включая Шумер, Египет, Византию, Китай, ацтеков, инков и основные религиозные традиции.
3. Работают поиск, фильтры, диапазоны, год, карточки, тема, масштаб, URL-состояние и CSV.
4. Проверки проходят локально и в GitHub Actions.
5. Репозиторий `agent-axiom/parallel-worlds` содержит ветку `main`, а Pages доступен по `https://agent-axiom.github.io/parallel-worlds/`.
